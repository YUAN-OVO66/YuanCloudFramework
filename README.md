# YuanCloudFramework
基于springCloud-Alibaba建立的微服务快速搭建框架



## 技术选型

- jdk 17
- springboot 3.1.4
- springcloud 2022.04
- springcloud-alibaba 2023.0.1.0

## 端口占用
- 8080：gateway网关
- 8081：cloud-core核心模块
- 8082：cloud-auth认证中心模块
- 8083：cloud-user用户模块
- 8084：cloud-ai人工智能模块
- 8085：cloud-admin管理模块

- 8848：nacos注册中心
- 8858：sentinel微服务治理
- 8868：skywalking可观测
